Google Chrome developer tools is found by clicking on settings on the top right corner of any web page, followed by "more tools" and "developer tool"The page will then split in two, either vertically or horizontally. One half is the opened web page and the other half is where you can inspect elements of the codes used for that page and you can play around with the tools available.When you hover around the code, chrome tells you where the table content is by highlighting it in blue on the page. Another way of knowing where certain code is referring to is by right clicking on the part of the website you want to inspect and select "inspect". It will take you directly to the part of the code that belongs to the highlighted area on the website.
You can also right click on a code an select "scroll to vie", it should take you to the exact part of the website that the code refers to.

Lesson 2.9 - HTML, CSS and DOM

How the browser knows what style to apply and where that styles comes from

HTML is the language of the web and it has specific sintax and rules, you write HTML in a document following HTML sintax and rules.
The basic word for HTML is tag. The browser turns HTML tags into elements that form a tree. It knows how to do that because of the Document Object Model (DOM) which is a standard convention for representing and interacting with elements in HTML.
The tree in a development tool is a DOM tree that the browser built from an HTML document. Each HTML tag creates an element in the DOM that the browser uses to display the page.
An element is created from the starting tag to the ending tag and everything in between is content. Example: (opening tag) <g;tag> content </tag> (closing tag).
The "content" can be empty, can be text or it can be another element. Each tag has a name or type that defines what kind of element will be created i.e. element can be a paragraph, an image or just a document division.
Block - Each tag can also have attibutes with values i.e. <h1 class="article"> where "class" is the attribute and "article" is the value of the attribute.
CSS allow you to use specific sintax and rules to change how the elements look on the page, the size of the font, colour, background, borders, position and so on. For example, CSS can be used to say that all h1 elements that belongs to class article should have a text colour green <h1 class="article">
To tie style and structure together, the HTML document can provide the browser with information on what style to use and where to get it. To action that, you will need to open a tab on your text editor titled index.html and another titled style.css. You will then write your html code under the index.html tab and style your text under the style.css tab.
